# Chimera Exchange Token (CEXT)

Smart contract repository for the CEXT token.


## Initializing the project

```
bash setup.sh
```

## Testing the contract

```
forge test
```

## Deploying the contract

You can simulate a deployment by running the script:

```
forge script script/ChimeraExchangeToken.s.sol
```

See [Solidity scripting guide](https://book.getfoundry.sh/guides/scripting-with-solidity) for more information.
